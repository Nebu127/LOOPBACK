# /project:fix-issue

Fix de un issue específico.

## Uso

```
/project:fix-issue <issue-id>
```

## Descripción

- Recibe un issue ID de GitHub
- Analiza el problema
- Propone y aplica solución
- Crea PR si aplica

## Permisos

- Read, Edit, Glob, Grep, Bash
- gh issue, git checkout, git commit, git push
