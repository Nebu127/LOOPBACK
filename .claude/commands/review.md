# /project:review

Review del código actual.

## Uso

```
/project:review
```

## Descripción

- Revisa los cambios pendiente en el git
- Verifica lint y tests
- Devuelve un resumen con problemas encontrados
- Sugiere fixes si es necesario

## Permisos

- Read, Glob, Grep, Bash
- git diff, git status
